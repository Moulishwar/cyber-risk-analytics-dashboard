# Cyber Risk Analytics Dashboard (Power BI)

## Overview
This project simulates a cyber risk culture monitoring system, focusing on user behaviour, phishing susceptibility, and training effectiveness.

## Key Features
- Phishing click and reporting rate analysis
- User risk scoring based on behaviour and training
- Risk categorisation (High / Medium / Low)
- Device and browser-based risk insights
- Interactive dashboard with filters and KPIs

## Tools Used
- Power BI
- Excel / CSV dataset
- DAX for calculations

## Key Insights
- Identified high-risk users based on phishing behaviour
- Observed variation in risk across devices and browsers
- Highlighted impact of training on reducing user risk

## Purpose
To simulate governance-style reporting and support data-driven cyber risk decision-making.
